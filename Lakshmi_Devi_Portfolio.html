PROJECT 02
S3 Storage Cost Optimisation — Lifecycle Policies
Cost Optimisation
01.Analysed S3 bucket usage and found that old files were being stored in expensive storage classes indefinitely.
02.Identified which data was frequently accessed vs. rarely used by reviewing access patterns and CloudWatch metrics.
03.Configured S3 lifecycle rules to automatically move files older than 30 days to S3 Standard-IA (Infrequent Access).
04.Set additional rules to move archived data to S3 Glacier after 90 days for long-term, low-cost storage.
05.Added expiration policies to automatically delete data that had exceeded its retention period — removing the need for manual cleanup.
✅Outcome: Significant reduction in monthly S3 storage costs with zero manual effort — data moves and deletes itself on schedule.
